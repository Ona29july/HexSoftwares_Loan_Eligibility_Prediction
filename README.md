 HexSoftwares_Loan_Eligibility_Prediction
 Loan Eligibility Prediction 

HexSoftwares Data Science Internship Project

 Overview
A machine learning project to predict whether a loan application
will be Approved (Y)  or Not Approved (N) based on applicant details.

 Dataset
- 614 loan applications
- 12 features (income, credit history, education, marital status, etc.)
- Target: `Loan_Status` (Y = Approved, N = Not Approved)

 What This Project Does
1. Load & explore the dataset
2. Handle missing values (mode/median)
3. Exploratory Data Analysis — visualisations
4. Encode categorical variables (LabelEncoder)
5. Train/Test split (80/20)
6. Train 3 ML models:
   - Logistic Regression
   - Decision Tree
   - Random Forest
7. Evaluate with accuracy & confusion matrix

 How to Run
pip install pandas numpy matplotlib seaborn scikit-learn
python loan_eligibility.py

 Tools Used
- Python 3.x | Pandas | NumPy
- Matplotlib | Seaborn
- Scikit-learn
